# Weather App

A weather app, made with:

- [x] HTML
- [x] CSS
- [x] JS

## What info is shown

- Current temperature
- 7 day forecast
- UV Index
- Wind Status
- Sunrise and Sunset
- Humidity
- Visibility
- Air Quality

## Online View

View the App hosted on [Github Pages](https://gianelli99.github.io/weather_app/).

## How to use it

1. Type a city
2. Choose one suggestion with a click
